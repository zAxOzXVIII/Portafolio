# Prompt del proyecto — Landing Page Portafolio

Usa este prompt para continuar el desarrollo con IA o para documentar el alcance del sitio.

---

## Prompt base

```
Crea y mantén una landing page profesional de portafolio para Marden Adrian BARRERA ROSALES.

Stack: HTML, CSS y JavaScript puro (sin frameworks).
Hosting: GitHub Pages (repositorio zAxOzXVIII/Portafolio).
Idiomas: bilingüe ES/EN con selector de idioma.
Estilo: tema oscuro, moderno, minimalista. Colores negro, blanco mate y acentos neutros.

Secciones obligatorias:
1. Hero — foto de perfil, nombre, título, frase y CTAs (CV, contacto, GitHub)
2. Habilidades — agrupadas por categoría (Backend, Frontend, DB, DevOps)
3. Experiencia — internet4E (sep 2025–actualidad), freelance (2022–actualidad)
4. Proyectos — tarjetas con descripción y tecnologías
5. Educación — Ingeniero en Informática + Ciberseguridad (en progreso)
6. Contacto — enlaces a GitHub, LinkedIn y WhatsApp

Fuente de datos: data/portfolio.json
Imagen de perfil: assets/images/profile.png
CV descargable: assets/cv/Marden_Barrera_CV.pdf

Requisitos técnicos:
- Responsive (mobile-first)
- Accesible (semántica HTML, contraste, aria-labels)
- Animaciones sutiles (scroll reveal, hover)
- Sin dependencias externas pesadas
- SEO básico (meta tags, Open Graph)
- Performance: imágenes optimizadas, CSS/JS minificables

No inventar datos. Si falta información, usar placeholders marcados como [PENDIENTE].
```
